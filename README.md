# TaskFlow – Task Management App

A modern, fully interactive task management application built with **React 18**, **TypeScript**, **Tailwind CSS v4**, and **Framer Motion**. Designed for teams and individuals to track tasks, time, projects, and collaborate in real time.

## ✨ Features

- 📊 **Dashboard** – Overview of tasks, time tracking, productivity stats, and recent activity.
- ⏱️ **Time Tracker** – Start/pause/stop timers per task, view weekly hours and category breakdown.
- 📅 **Calendar** – Visualise tasks and events on a monthly/weekly calendar.
- ✅ **My Tasks** – Full task list with filters (status, priority, due date) and search.
- ➕ **Task Creation** – Create/edit tasks with title, description, category, priority, due date, tags, attachments.
- 📁 **Projects** – Group tasks into projects, track progress per project.
- 👥 **Team** – Manage team members, roles, and permissions.
- 💬 **Chat** – Real‑time team messaging (UI ready; can connect to a backend).
- ⚙️ **Settings** – Profile settings, appearance (primary colour, background, surface colour, border radius) – saved in localStorage.
- 🎨 **Dark Theme** – Fully responsive dark UI with custom CSS variables.
- 🖱️ **Smooth Animations** – Hover effects, transitions, and page transitions using Framer Motion.

## 🛠️ Tech Stack

- [React](https://react.dev/) – UI library
- [TypeScript](https://www.typescriptlang.org/) – Type safety
- [Tailwind CSS v4](https://tailwindcss.com/) – Utility‑first styling
- [Framer Motion](https://www.framer.com/motion/) – Animations
- [React Icons (Feather)](https://react-icons.github.io/react-icons/) – Icon set
- [React Router DOM](https://reactrouter.com/) – Routing (for multiple pages)
